BEFORE YOU DO ANYTHING, YOU MUST HAVE RAWTHERAPEE VERSION 5.10 OR LATER INSTALLED.

Copy the contents inside the 1 - PP3 Files to Install folder to:

Windows:
C:\Program Files\RawTherapee\profiles
(you need to have administrator privileges)

Linux:
usr/share/rawtherapee/profiles
(you need to have root privileges like using "sudo nautilus" in Fedora or Ubuntu for example - please note that you might have to either copy an existing folder inside this directory & rename it & then paste the pp3 files into them or "chmod" a folder to get the correct settings so that RawTherapee can run)

The contents inside of 2 - DCP Files Installation have the installation instructions inside.

The contents inside of 3 - Optional dynamicprofile Installation are completely optional. The installation info can be found there.

The contents inside of 4 - Additional DCP files are completely optional. The installation info can be found there.
