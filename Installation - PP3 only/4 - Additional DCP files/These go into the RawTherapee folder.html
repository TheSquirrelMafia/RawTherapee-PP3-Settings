These are the default Pentax Bright .dcp files for your camera.

This is entirely optional.

Copy the contents inside the dcpprofiles folder to:

Windows:
C:\Program Files\RawTherapee\<whateverversionyouhave>\dcpprofiles
(you need to have administrator privileges)

Linux:
usr/share/rawtherapee/dcpprofiles
(you need to have root privileges like using "sudo nautilus" in Fedora or Ubuntu for example)

It will ask you to overwrite previous DCP files. Make sure you’re certain about overwriting files ‘cause you may lose a custom .dcp file that you’ve already made.
